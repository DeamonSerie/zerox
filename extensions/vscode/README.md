# Zerox Language Extension

Syntax highlighting and basic editor configuration for Zerox `.0` files in VS Code and Cursor. The repository also ships `pnpm run zls -- --self-test`, which exercises diagnostics, hovers, completions, symbols, fix plans, and editor metadata from `zerox graph`, `zerox size`, `zerox mem`, and `zerox doc`.

Install into Cursor from the repository root:

```sh
pnpm run extension:install:cursor
```
